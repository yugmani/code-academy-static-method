Classes: Static Methods

  • The static keyword is used to create a static method in the parent class.
  • Syntax: 
    class ClassName {
  constructor(){
    //...
  }
    
  static functionName(){
    
    // ...
    
  }
}
  • We call it(static method) directly from the class, but not from an instance of the class. 
    console.log((parentClass.methodName()));
  • You cannot access the static method from instances of the parent class or instances of its child classes (See below). 
    const myInstance = new parentClass("argument");
    
myInstance.staticMethod();  //TypeError;
